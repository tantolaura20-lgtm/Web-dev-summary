1. HTML Basics & Structure

What i learned:
HTML (HyperText Markup Language) is the standard language for creating web pages. A basic HTML document has a doctype, html, head, and body section

Example:
<!DOCTYPE html>
<html>
<head>
  <title>My Page</title>
</head>
<body>
  <h1>Welcome!</h1>
  <p>This is a simple page.</p>
</body>
</html>


2. HTML Elements

What i learned:
Elements are building blocks of HTML, consisting of a start tag, content, and an end tag. Some elements are empty (no end tag) 

Example:
<p>This is my first paragraph.</p>
<br> <!-- Empty element -->


3. HTML Attributes

What i learned:
Attributes provide extra information about elements. They usually come in name="value" pairs inside the opening tag

Example:
<a href="https://example.com">Visit Example.com</a>
<img src="logo.png" alt="Logo" width="100">


4. HTML Headings

What i learned:
Headings (<h1> to <h6>) define titles and structure content. <h1> is the most important, <h6> the least

Example:
<h1>My Title</h1>
<h2>This is a subsection</h2>


5. HTML Paragraphs and Text

What i learned:
Paragraphs use <p>. HTML ignores extra spaces/line breaks; use <br> for a line break

Example:
<p>This is paragraph one.</p>
<p>This is paragraph two.</p>
<p>This a line<br>breaks here.</p>


6. HTML Links

What i learned:
Use the <a> tag to create clickable links. The href attribute defines the destination. 

Example:
<a href="https://w3schools.com">Go to W3Schools</a>


7. HTML Images

What i learned:

8. HTML Lists

What i learned:
Ordered (<ol>) and unordered (<ul>) lists group related items with <li>

Example:
<ul>
  <li>Item A</li>
  <li>Item B</li>
</ul>

<ol>
  <li>First</li>
  <li>Second</li>
</ol>


9. HTML Tables

What i learned:
Tables organize tabular data with <table>, <tr> (rows), <th> (headers), and <td> (cells). 

Example:
<table>
  <tr><th>Name</th><th>Age</th></tr>
  <tr><td>Alice</td><td>30</td></tr>
</table>


10. HTML Forms

What i learned:
Forms (<form>) collect user input with elements like <input>, <label>, checkboxes, radio buttons, and submit buttons. 

Example:
<form>
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <button type="submit">Send</button>
</form>
