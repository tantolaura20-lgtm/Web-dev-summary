HTML Summary
HTML is a markup language used to structure content on a webpage.

Every element has an opening tag, content, and a closing tag. 
Ex:<p>This is my first page.</p>

Attributes provide extra information and are always inside the opening tag. Ex: <img src="photo.jpg" alt="My Photo">

HTML has 6 heading levels (h1 to h6). Ex: <h1>Main Title</h1>

Paragraphs are sed to write blocks of text.
Ex: <p>This is a paragraph.</p>

Links are created with the <a> tag and use the href attribute.
Ex:
<a href="https://example.com">Visit Site</a>

Images use the <img> tag. It has no closing tag.Example:

<img src="image.jpg" alt="A sample image">

HTML supports ordered and unordered lists.
Example:

<ul>
  <li>Milk</li>
  <li>Bread</li>
</ul>

<ol>
  <li>Step one</li>
  <li>Step two</li>
</ol>

Divs & Sections are Used to group content.
Ex:

<div>
  <h2>My Section</h2>
  <p>Some text here.</p>
</div>

Forms:Used to collect user information.

Ex:
<form>
  <input type="text" placeholder="Your Name">
  <input type="email" placeholder="Email">
  <button>Submit</button>
</form>
